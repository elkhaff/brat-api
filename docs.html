Brat API

Generate gambar & video bergaya "brat" (album cover Charli XCX) dari teks apa aja — buat bot, project, atau eksperimen lo.

by elkaff · github.com/elkhaff/brat-api

Endpoints

GET/img

Generate gambar PNG 500×500 statis.

ParamTipeDefaultKeterangan
textstringbratTeks yang ditampilkan. Gak ada batasan panjang.
backgroundhex color#ffffffWarna background, contoh %23ff0000 (urlencode #).
colorhex color#000000Warna teks.

Contoh

GET /img?text=brat+summer&background=%2300ff66&color=%23000000

GET/vid

Generate video MP4 500×500 — teks muncul kata demi kata, mirip animasi di bratgenerator.com.

ParamTipeDefaultKeterangan
textstringbratTeks yang dianimasiin. Gak ada batasan panjang.
backgroundhex color#ffffffWarna background.
colorhex color#000000Warna teks.
speedinteger (ms)500Jeda antar kata muncul. Dibatasin 200–2000.
holdinteger (ms)1200Lama frame terakhir ditahan sebelum video selesai. Dibatasin 0–5000.

Contoh

GET /vid?text=brat+and+it%27s+completely+different&speed=400&hold=1500

Cache

Setiap kombinasi parameter di-hash pakai SHA256 dan hasilnya disimpan di folder /cache di server. Request berikutnya dengan parameter yang persis sama langsung diambil dari cache (response header X-Cache: HIT), gak perlu render ulang lewat browser. Kalau parameternya beda dikit aja (termasuk speed/hold), hash-nya beda → generate baru (X-Cache: MISS). Cache otomatis kedaluwarsa setelah 30 hari.

Font Emoji

Server udah ter-install font Apple Color Emoji (build Linux) lewat Docker image, jadi emoji di dalam text ke-render dengan benar (bukan kotak kosong).

Error Response

Kalau ada masalah pas generate, response berupa JSON:

{
  "author": "elkaff",
  "repository": { "github": "https://github.com/elkhaff/brat-api" },
  "message": "Gagal membuat video",
  "error": "..."
}
